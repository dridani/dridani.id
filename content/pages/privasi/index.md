**dridani.id** adalah blog/website yang sepenuhnya milik **Dony Ridani** sendiri, dan bukan bagian dari suatu perusahaan ataupun organisasi manapun. Halaman ini, akan menjelaskan bagaimana saya menggunakan data kamu pada saat kamu mengunjungi website ini.

## Pihak ketiga

**dridani.id** tidak akan memperjualbelikan data pribadi kamu, dan tidak terhubung dengan layanan pihak ke-*tiga*, selain layanan  **Domain**, **Hosting**, dan **Google Search Console**.

## Cookies

Website ini menggunakan *cookies*. **Cookies** adalah file kecil yang ada pada suatu website untuk mentransfer data kedalam perangkat kamu melalui Web Browser (hanya jika kamu mengizinkannya).

## Link / Tautan

Mungkin terdapat link pada suatu artikel di website ini. Link Website yang terdapat di website ini memiliki Kebijakan Privasi tersendiri. Maka dari itu, saya tidak bertanggungjawab atas aktifitas tersebut.

## Pembaruan 

Saya berhak untuk memperbarui kebijakan privasi ini kapan saja tanpa pemberitahuan.
